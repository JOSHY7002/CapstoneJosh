ALGEBRA AND EQUATIONS LESSON

What will you learn in this lesson?

In this lesson you will learn how to solve 1 step, and multi step equations.

1 step equations

The main purpouse in solving equations is to ISOLATE the variable that is being solved

Example:1

Find X: x+4 = 6.
In equations you have to try to find a way to isolate the variable. To isolate a variable you must get rid of the iteger next to it using oppsite operations.
In this case since since its x + 4 you have to subtract 4 from BOTH sides, x + 4 -4 = 6 - 4. Meaning x = 2

Example:2

Find X: x(5)=15
In this case the operation next to the variable is multiplication so the opposite operation is divsion.
x(5)/5 = 15 / 5. meaning x = 3

Example:3

Solve for x: 12 = x/4
In this case the operation next to the variable is divsion so the opposite operation is multiplication.
12 * 4 = x/4 * 4. meaning x = 48

Multi step equations

In multi-step equations, like the name states there are multiple steps in these equations.

2 step equation

Find X: x4 + 7 = 23.
In this equation, again you have to find a way to isolate the variable.
First: Look at the 7 that is next to the variable and use opposite operations to remove it, 4x + 7 - 7 = 23 - 7
Then you get 4x = 16, now this is a one step equation, 4x / 4 = 16 / 4, therefore x = 4. See Easy!

Variables on both sides.

Find X: 3x + 4 = 4x - 7
In this case you must remove one of the variables on either side, if they are both positive choose the smaller one.
First: 3x - 3x + 4 = 4x - 3x - 7, When cancelling out variables only subtract it from another variable.
Second: 4 = x - 7, now its a one step equation! x = 4 + 7, x = 11.

Distributive property

In the quiz you will have to deal with some problems involving distributive property

Example: 1

3(3+x)
Distributive property is when you see an integer beside a bracket with numbers within it you must multiply that integer with everything within it.
In this case 3 ( 3 + x ) = 3 *3 + 3 * x, and that equals 9 + 3x.

Here you have a negative number, so you must account for the negative nummbers.
-2 (4x - 5) = -2 * 4x (-2)*-5, and that equals -8x + 10(because negative times a negative is a positive).
