EXPONENT RULES LESSON

What exactly is exponents?

In short, power or exponent indicates the number of times a number needs to be multiplied by itself. Here, the base can be any integer, fraction or decimal. The exponent can also take up any value, be it positive or negative.

This is what an exponent is:

3^3 means 3 x 3 x 3 = 81

Another example:

5^4 means 5 x 5 x 5 x 5 = 625

In this lesson, you will be taught the basic rules of exponents

Like anything else in math, exponents have different rules when different situations happen.

Product Rule: Multiplying exponents

a^m * a^n = a^m + n. You ADD the exponents together

Example:

3^5 * 3^4 = 3^ 5 + 4 = 3^9

Quotient rule Rule: Dividing exponents

a^m / a^n = a^m - n. You SUBTRACT the exponents

Example:

6^7 * 6^2 = 6^ 7 - 2 = 3^5

Power rules:

(a^m)^n = a^m * n. You MULTIPLY the two exponents

Example:

(2^5)^5= 2^ 5 x 5 = 2^25

Negative exponents

A number with a negative exponent should be put to the denominator, and vice versa

Example:

5^ -7 just means 1/5^7

Zero rule

A number with an exponent that is 0 is ALWAYS equal to one.

Example:

8^0 = 1 or 236273^0

One rule

A number with an exponent that is 1 is just the base number.

Example:

5^1 =5 or 343^1 = 343
