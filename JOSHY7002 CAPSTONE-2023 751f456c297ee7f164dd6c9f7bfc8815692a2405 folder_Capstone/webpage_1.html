Exponents

You will learn the basic exponential rules using videos, examples and explanations, so you can do a simple quiz on them. You will learn how to multiply, simplify and divide exponents in this section

Equations

You will learn how to solve simple linear equations rules using videos, examples and explanations, so you can do a simple quiz on them.

Mental-Math

In this section you will learn multiple tricks, so you can multiply, divide, add and subtract better in your head. Then you will do a simple quiz. You are not allowed to use a calculator!
